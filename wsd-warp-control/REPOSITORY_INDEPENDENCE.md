# Repository Independence

This repository is the independent private repository for the WSD Warp Control lane.

Ownership follows the true project, entity, lane, and owner assignment. It must not be renamed, organized, or governed around a tool unless that tool is the actual project.

It is not subordinate to `billydeeii136/Cloudflare`, and it must not be merged into Cloudflare as a master repository.

Cloudflare, GitHub, terminal sessions, AI assistants, and other tooling are service or integration layers only. They do not own this repository and do not define its naming convention.

Federated coordination records may reference this repository, but coordination does not transfer ownership.

Do not commit secrets, API tokens, SSH keys, private keys, account credentials, `.env` files, or credential-vault material.
